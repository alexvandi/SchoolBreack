# QR Card Management System

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) with your browser.

## Features
- **Admin Dashboard**: Create and manage promotions.
- **Shop Scanner**: Secure PIN (`1234`), QR Scanner, and Verify Logic.
- **Card Activation**: User registration form.

## Troubleshooting
- If you encounter CSS errors during build, ensure you are using Node.js 18+ and have `postcss` configured correctly for Tailwind v4.
- The project uses Next.js 16 and Tailwind 4.
